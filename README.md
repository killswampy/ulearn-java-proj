Проект Ulearn java

Дан файл data.csv в котором находится отчет о продажах различных стран
0. Создал набор моделей по CSV-файлу
1. Распарсил данные и создал объекты в java используя библиотеку OpenCSV
2. Создал БД SQLite и подключился к ней. В ней создал таблицу Orders, куда поместил все обьекты с помощью SQL-запросов используя библиотеку SQL и SQLite
3. Построил график
3.1 Получил данные в нужном виде используя SQL запрос
3.2 Полученные данные визуализировал используя библиотеку JFreeChart
4. Получил лидера по продажам через SQL-запрос и вывел в консоль
5. Получил список стран в африке у которых доход в промежутке 420 000 - 440 000 используя SQL-запрос и вывел в консоль
